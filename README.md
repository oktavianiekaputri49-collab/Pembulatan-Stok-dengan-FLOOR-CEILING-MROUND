# Pembulatan-Stok-dengan-FLOOR-CEILING-MROUND
Mini project menggunakan Microsoft Excel untuk mengelola data stok dengan fungsi FLOOR, CEILING, dan MROUND. Pembulatan digunakan untuk menentukan jumlah stok yang dibutuhkan. Project ini melatih pengolahan data dan pengambilan keputusan sederhana dalam manajemen persediaan.
